
# 📋 API NLW Journey

Backend do repositório "FE - NLW Journey".

Esse projeto foi criado para ser consumido por um frontend. Nessa aplicação será possível criar e editar viagens, participantes poderão consultar atividades da viagem e confirmar ou não sua presença.

Essa API foi criada utilizando node.js, fastify e prisma. O banco da dados utilizado foi o SQLite.

# ✍🏻 Como instalar

Clone o projeto, utilizando o terminal:

```bash
  git clone https://github.com/mariquegonn-dev/api-nlw-journey
```

Instale as dependências:
```bash
  npm install
```

Configure as variavéis de ambiente, utilize o .env.example
```bash
  DATA_BASE_URL="ENDEREÇO DO BANCO"
  API_BASE_URL="ENDEREÇO QUE A API ESTÁ RODANDO"
  WEB_BASE_URL="ENDEREÇO QUE O FRONT END ESTÁ RODANDO"
  PORT=NUMBER
```

Para iniciar o projeto:

```bash
  npm run dev
```
